# MonC - Monitoring CCTV System

**MonC (Monitoring CCTV)** is an enterprise-grade web-based CCTV monitoring system designed for customs facilities, warehouses, and multi-building security operations.

## Quick Start

See INSTALL.md for detailed installation instructions.

## Features

- Multi-building camera management (256 cameras across 4 buildings)
- Real-time live monitoring with multiple grid layouts
- Video playback and export
- AI-powered license plate recognition
- Watchlist management and incident tracking
- Comprehensive audit logging
- Role-based access control

## Tech Stack

- Laravel 12 + PHP 8.3+
- MySQL 8.0+
- FFmpeg for video streaming
- Python FastAPI for AI analytics
- Tailwind CSS + Alpine.js

Full documentation available at: https://docs.monc.local
